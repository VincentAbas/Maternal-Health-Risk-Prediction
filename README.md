## Data Info

Age = User/Patient's Age
SystolicBP = Systolic Blood Pressure
DiastolicBP = Diastolic Blood Pressure
BS = Blood Sugar
BodyTemp = Body Temperature
HeartRate = Heart Rate
Risk = Risk Level (Low Risk, Med Risk, High Risk)
